During Compile -
1. Go to KBM-Robot module and open up the main js file.
2. Edit the java path under startJar to point to root java folder.
3. Run build.
4. Copy java folder and paste it into the root of the build folders.

Rebuild Robotjs -
1. set PYTHON=D:\Python\bin\Python.exe
2. Run "node-gyp rebuild --directory=node_modules/robotjs/ --runtime=electron --target=1.4.1 --arch=x64 --dist-url=https://atom.io/download/atom-shell"
